# Vicunav Standards

Este repositorio reúne los estándares técnicos compartidos del ecosistema Vicunav y sirve como fuente común para los demás repositorios mediante un submódulo.
Está dirigido a Codex, al responsable del ecosistema Vicunav y a cualquier colaborador futuro que diseñe, implemente o revise cambios.
La documentación vive en `docs/` y debe consultarse antes de tomar decisiones transversales.
